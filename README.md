# Osama Rashid
### Senior QA Engineer / SDET

I don't just write test cases — I build the systems that make quality scalable. Most of my work lives at the intersection of test automation and internal tooling: frameworks that teams can actually maintain, CI pipelines that catch regressions before they ship, and scripts that remove the manual, repetitive parts of QA so testers can focus on judgment calls instead of busywork.

A recent example of that last part: an AI-powered tool that reads Azure DevOps user stories and surfaces structured QA insights — acceptance-criteria gaps, missing edge cases — before a story ever reaches a sprint. On the automation side, I work across Playwright and Cypress for UI, API, and BDD-style testing, usually built around Page Object Model and wired into Azure Pipelines rather than left to run manually.
