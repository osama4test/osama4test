# Hi there! 👋 I'm Osama Rashid

### Senior QA Engineer | SDET

I don't just write test cases — I build the systems that make quality scalable. Most of my work lives at the intersection of test automation and internal tooling: frameworks that teams can actually maintain, CI pipelines that catch regressions before they ship, and tools that remove the manual, repetitive parts of QA so testers can focus on judgment calls instead of busywork.

---

### 🔍 What I Do

- **Building automation frameworks** — Playwright and Cypress suites structured around Page Object Model, covering UI, API, and BDD-style testing rather than one-off scripts.
- **AI-assisted QA tooling** — built a tool that reads Azure DevOps user stories and surfaces structured QA insights (acceptance-criteria gaps, missing edge cases) before a story reaches a sprint.
- **API & BDD testing** — Cypress + Gherkin-driven REST API test suites for behavior-driven coverage.
- **CI integration** — wiring suites into Azure Pipelines so regressions get caught automatically, not run by hand.
